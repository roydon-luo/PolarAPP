# Checkpoints

Pretrained checkpoints are distributed separately through the project page or a link provided by the authors.

Place the checkpoint files as follows:

```text
experiments/checkpoints/
  DemNet/
    DemNet.pth
  TaskNet/
    TaskNet.pth
  FANet/
    FANet.pth
```

Training updates these fixed filenames and stores the current epoch inside each checkpoint.
