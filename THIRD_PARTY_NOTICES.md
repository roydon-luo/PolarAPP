## PolarFree and PolaRGB

The DfP TaskNet is based on [PolarFree](https://github.com/mdyao/PolarFree),
and the DfP experiments use the
[PolaRGB dataset](https://huggingface.co/datasets/Mingde/PolaRGB). PolaRGB is
published as CC BY-NC 4.0 and distributed through its original dataset page.

At the time this release candidate was assembled, the public PolarFree GitHub
repository lacked a discoverable `LICENSE` file. Obtain written redistribution
permission from the PolarFree copyright holders before publishing the
DfP-derived source under the project license. An appropriately licensed clean
implementation provides an alternative release path.

## SfPUEL data

The SfP dataset links come from
[SfPUEL](https://github.com/YouweiLyu/SfPUEL). Follow the terms provided by the
dataset authors and cite SfPUEL when using their data.

## Restormer-derived building blocks

Several transformer building blocks follow the Restormer architecture. Keep
the upstream attribution and MIT notice when the final provenance review
identifies files derived from the
[official Restormer implementation](https://github.com/swz30/Restormer).
